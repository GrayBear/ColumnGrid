ColumnGrid
==========

Responsive CSS only column grid.

##How to use

    <div class="columns">
      <div class="cell">
        
        <p>first</p>
        
      </div>
      <div class="cell">
        
        <p>second</p>
        
      </div>
      <div class="cell">
        
        <p>third</p>
        
      </div>
    </div>

This will create

On Large Screens

    -------------------------------------
    | first    | second     | third     |
    -------------------------------------
